# 🏥 Hospital Management System

Современное desktop-приложение для учёта пациентов, посещений и занятости палат, разработанное на базе **Qt 6**, **QML** и **SQLite**.

---

## 📌 О проекте

Приложение предназначено для автоматизации работы медицинского учреждения:

- регистрация сотрудников;
- авторизация пользователей;
- учёт пациентов;
- регистрация посетителей;
- просмотр истории посещений;
- контроль занятости палат.

Интерфейс реализован на **QML** с использованием кастомных компонентов, а вся логика и работа с базой данных написана на **C++**.

---

## ✨ Основные возможности

### 🔐 Авторизация и регистрация

- создание новых аккаунтов сотрудников;
- безопасное хранение паролей;
- SHA256 + соль для хеширования.

### 🧾 Управление пациентами

- добавление пациентов;
- хранение ФИО, диагноза, номера палаты и паспортных данных;
- отображение списка пациентов.

### 🚪 Учёт посещений

- регистрация посетителей;
- привязка посещения к пациенту;
- сохранение времени визита и ответственного сотрудника.

### 📋 История посещений

- просмотр всех зарегистрированных посещений;
- подробная информация о визите;
- фильтрация и навигация по данным.

### 🏠 Контроль палат

- отображение занятости палат;
- группировка пациентов по палатам;
- быстрый просмотр информации.

### 🎨 Интерфейс

- современный UI на QML;
- тёмная цветовая схема;
- анимированный фон;
- кастомные кнопки и поля ввода;
- бесрамочное окно;
- возможность перетаскивания окна мышью.

## 🚀 Сборка проекта

### 1. Клонирование репозитория

```bash
git clone https://github.com/geritoda1/hospital.git
cd hospital
```

### 2. Открытие проекта

Откройте файл `CMakeLists.txt` через Qt Creator.

### 3. Настройка компилятора

Выберите необходимый Kit.

### 4. Сборка

```text
Build → Build All
```

После сборки приложение появится в папке:

```text
build/
```
